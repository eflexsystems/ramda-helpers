assocDp(path, val, obj) → {Object}

Source:
  • index.js, line 77

Makes a shallow clone of an object, setting or overriding the nodes required to create the given dotted path, and placing the specific value at the tail end of that dotted path. Note that this copies and flattens prototype properties onto the new object as well. All non-primitive properties are copied by reference.

Example
assocDp('a.b.c', 42, {a: {b: {c: 0}}}); //=> {a: {b: {c: 42}}}

// Any missing or non-object keys in path will be overridden
assocDp('a.b.c', 42, {a: 5}); //=> {a: {b: {c: 42}}}
Parameters:
Name Type Description

The dotted path to set
val *

The new value
obj Object

The object to clone
Returns:

A new object equivalent to the original except along the specified dotted path.

dp(path, obj) → {*}

Source:

Retrieve the value at a given nested dotted path.

Example
dp("a.b", {a: {b: 2}}); //=> 2
dp("a.b.", {c: {b: 2}}); //=> undefined
dp("a.b.0", {a: {b: [1, 2, 3]}}); //=> 1
dp("a.b.-2", {a: {b: [1, 2, 3]}}); //=> 2

dpEq(path, val, obj) → {Boolean}

Source:
  • index.js, line 56

Determines whether a dotted path on an object has a specific value, in R.equals terms. Most likely used to filter a list.

Example
const user1 = { address: { zipCode: 90210 } };
const user2 = { address: { zipCode: 55555 } };
const user3 = { name: 'Bob' };
const users = [ user1, user2, user3 ];
const isFamous = dpEq('address.zipCode', 90210);
R.filter(isFamous, users); //=> [ user1 ]
Parameters:
Name Type Description
path String

The dotted path of the nested property to use
val *

The value to compare the nested property with
obj Object

The object to check the nested property in
Returns:

true if the value equals the nested object property, false otherwise.

dpPluck(key, f) → {Array}

Source:
  • index.js, line 96

Returns a new list by plucking the same dotted path off all objects in the list supplied.

pluck will work on any functor in addition to arrays, as it is equivalent to R.map(dp(k), f).

Example
dpPluck('name.first', [{name: { first: 'fred' }}, {name: { first: 'wilma'} }]); //=> ['fred', 'wilma']
Parameters:

leftJoin(leftComparedPath, rightComparedPath, rightCollection, leftCollection) → {Array}

Source:
  • index.js, line 156

Returns a new list by where each leftCollection object has its leftComparedPath replaced with the object from rightCollection that has a rightComparedPath value equal to the value at leftComparedPath

Example
const leftCollection = [{ a: "a", b: 2 }];
const rightCollection = [{ id: "a", c: 2 }];
const result = leftJoin("a", "id", rightCollection, leftCollection); //=> [{ b: 2, a: { id: "a", c: 2, } }]
